# CovidAnalyticsPySpark
This project focuses on building an end-to-end data analytics pipeline using PySpark to analyze COVID-19 trends using multiple real-world datasets. The datasets contain information such as daily confirmed cases, deaths, recoveries, active cases, country-wise statistics, WHO region data, population details, and county-level geographic information.

## Tech Stack
- PySpark
- Python
- Spark SQL
- Pandas
- Matplotlib
- Plotly

## Datasets
- full_grouped.csv
- covid_19_clean_complete.csv
- country_wise_latest.csv
- day_wise.csv
- usa_county_wise.csv
- worldometer_data.csv
